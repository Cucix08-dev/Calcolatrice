Calcolatrice scientifica che puo' calcolare: coseno, seno, tangente, cotangente, cosecante, secante, logaritmo in base 10, puo' elevare la potenza, puo' sommare/sottrare/moltiplicare/dividere numeri.
Tutto come una calcolatrice normalissima, semplicemente puo' cambiare lo stile della calcolatrice da modalita' Dark a modalita' Light, ovvero cambiare il theme della calcolatrice.
